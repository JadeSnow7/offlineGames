### Description

Offline Games 将六款经典与原创小游戏整合到一个简洁、流畅、注重隐私的应用中。

包含游戏：
- Snake（贪吃蛇）：不断变长，躲避边界，挑战最高分。
- Block Puzzle（方块拼图）：下落与旋转方块，完成消行。
- Breakout（打砖块）：控制挡板击碎砖块，保护生命值。
- Minesweeper（扫雷）：揭示安全格并标记地雷，完成全图。
- Memory Match（记忆配对）：翻牌并配对全部卡片。
- Reaction Tap（反应点击）：在正确时机点击，测试反应速度。

核心体验：
- 全程离线可玩。
- 无广告、无内购、无需账号。
- 支持英文与简体中文。
- 每款游戏都有本地最高分。
- 支持暂停、重开与快速继续。
- 提供音效与触感开关。
- 面向 iPhone 竖屏优化。

隐私声明：
Offline Games 不收集、不共享、不追踪任何个人数据。游戏设置和分数仅保存在你的设备本地。
